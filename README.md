### 💎 Projeto em Destaque: Library Manager (Concluído)
Este é o meu projeto "flagship", onde apliquei maturidade técnica para resolver problemas complexos de gestão de acervos.

*   **O que é:** API REST para controle rigoroso de empréstimos, devoluções e multas.
*   **Diferenciais Técnicos:**
    *   **Abordagem TDD:** Desenvolvimento guiado pelo ciclo *Red-Green-Refactor* em todos os serviços.
    *   **Regras de Negócio:** Sistema inteligente de cópias e cálculo automático de multas (R$ 2,00/dia).
    *   **Arquitetura e Segurança:** Autenticação stateless via **JWT**, tratamento de exceções customizado (GlobalExceptionHandler) e filtros dinâmicos com **JPA Specification**.
*   **Stack:** Java 21, Spring Boot 3+, PostgreSQL, Docker e SpringDoc OpenAPI.
*   **[🔗 Explorar Repositório](https://github.com/Andrey479/librarymanager)**
